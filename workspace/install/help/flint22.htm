FLINT GEOMETRY DEFINITION

Before proceeding further please refer to the Important Concepts of Flint Geometry here .

Complete geometry and grid definition normally takes place by the following sequence of steps. However, Flint is inteligent enough to allow skipping some of the steps and jumping backward and forwards to change the geometry or grid features later on. Any step which is not possible to take yet does not appear in the Geometry-Setup menu, making it easy to navigate.

  1. DEFINE GEOMETRY
  2. REFINE GEOMETRY (optional)
  3. DEFINE GRID
  4. REFINE GRID (optional) and/or MANIPULATE GRID (optional)
  5. SET CELLS
