EXPERT MENU

All advanced aspects of CFD modelling relating to the boundary-conditions, physical models and solution strategies are gathered together into the EXPERT menu. Without the use of EXPERT options Flint will be operating in a 'default' mode with the advantage of reducing the number of user interactions needed. But this will also limit the type of problems that can be tackled.

Expert menu contains the following sub-menus classified according to their functionality.

  • BOUNDARY OPTIONS
  • SOLUTION PARAMETERS
  • SELECT VARIABLES
  • UNDERRELAX
  • PATH-DATA
  • PHYSICAL-MODELS
  • BODY-FORCES
  • SWEEPS OF SOLVER
  • PRESSURE SOLVER
