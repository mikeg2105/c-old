VIEW MENU

This menu item contains all the graphical visualisation tools provided by Flint to display the results of the CFD calculations. It contains the following commands or sub menus;

  • OUTLINE
  • GRID
  • VECTORS
  • CONTOURS
  • FILLED-CONTOURS
  • PROFILES
  • XYPLOTS
  • OPTIONS
The OUTLINE command draws the geometrical outlines of the domain, which is definable as the wall/live_cells boundary. This command is available as soon as the computational grid is fully defined by means of the SETUP menu or by reading in a CASE file.

GRID command draws the outlines of the computational grid by drawing the finite volume boundaries. This command is also available after the full definition of the computational grid.

VECTORS command displays the velocity vectors. There will be a single vector drawn for each computational(live) cell within the domain. These vectors will be colour coded according to the magnitude of the velocity. User is given the option to change the scaling or the colour coding of these vectors via the OPTIONS -> PARAMETERS menu.

CONTOURS and FILLED-CONTOURS menus work in the same way, drawing the equi-value contours for any selected field variable. The only difference between the two being that, in the case of filled-contours, the areas between the contours are filled with the appropriate colours.

PROFILES are used for the graphical display of the variations of a selected variable along a particular line. The line along which the profile will be displayed can be selected to be one of the following types;
  • Along I-Grid lines
  • Along J-Grid lines
  • Along the X-Axes
  • Along the Y-axes
  • Along arbitrary lines
For the first four options, user will be invited to enter the number of profiles he requires, so that a series of profiles will be displayed along the specified direction.
The case of arbitrary profiles work differently. Instead of being invited to enter the number of profiles required, the user will be invited to define a line 'by means of the mouse', along which the profiles will be displayed.
This is one of the few instances of the need for the mouse-drag operation in Flint. User will be expected to define the arbitrary line by clicking the mouse button down at one end of the line and dragging the mouse while keeping the mouse button down. Flint will display the line while this operation takes place so that it can be located to users satisfaction. When satisfied user can define the final end of the line by releasing the button and the profiles are displayed immediately along this line. There is no limit to the number of profiles that can be drawn on the same graph by repeating the process above.

XYPLOTS command can be used to draw XY plots of the selected set of flow-field variables along a specified line. Inputs for this option are by means of the keyboard rather than the mouse. That is to say the space coordinates of the lines must be entered manually.
The procedure for plotting involves the following steps;
  • Get-Data: This involves specifying a line and a list of variables which will be recorded over that line.
  • Chose-Data: Each set of variables picked-up by Get-Data is referred to as a TABLE. Chose data selects which table to use for the subsequent plotting operation. Tables are numbered from 1 onwards in the order of their creation.
  • Plot : Plot a selected list of variables from the currently chosen table. Variables are numbered according to the order of their selection during the GET-DATA stage above. Selecting more than one variable will allow plotting of multiple variables on the same plot.
  • Display-Table,Delete-Table,Write-Table,Options are auxiliary commands to help manage tables of data.

OPTIONS command controls a few of the graphics related parameters.
These are:
  • Rescale: This allows the user to attribute different scaling constants for X and Y directions, producing distortion effects. It may be of use when trying to view domains which may need to be expanded in one direction so as to be able to discriminate fine geometrical features.
  • Mirror View: This allows a mirror view of a graph to be shown alongside the actual view.It may be of use for the visualisation of axi-symetric problems.
  • Colour List: Allows the user to change some of the default colours such as the grid and outline colour.
  • Parameters: Graphics related parameters which include;
    • Vector Scaling factor: Exaggerate or reduce the arrow sizes when plotting vector graphs.
    • Draw Arrowheads: Controls drawing of vector heads.
    • Profile Scaling Factor: Exaggerate or reduce the height of the profiles when drawing profiles
    • Fill Profiles: Draws solid colour profiles by filling the area between the profile and the base of the profile
    • Set Profile range, Contour Range and Vector Scaling Range: These are usually automatically selected by Flint to cover all the values within the computational domain, but user may define them manually to investigate finer variations around a smaller range.
