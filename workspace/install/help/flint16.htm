BASIC STRUCTURE OF FLINT

The menu structure of FLINT reflects the steps that are needed to define and solve a CFD problem, which can be summarised in the following cronological order:

  1. Define the Pysical Geometry:SETUP
  2. Define the Computational Grid:SETUP-->GRID
  3. Define the Boundary Conditions:SETUP-->BOUNDARY
  4. Define Physical Properties of the media:SETUP-->PHYSICAL-CONSTANTS
  5. Solve the problem:SOLVE
  6. Display the results:VIEW
There are also the usual menu items relating to file operations for saving the geometry, grid and results on the computer so that they can be restored back into Flint or read into other packages later. These are all collected together under the heading of FILE main menu.

Finally all the advanced features of Flint relating to the problem definition or solver related features are presented together by means of the EXPERT menu.
