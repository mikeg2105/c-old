PHYSICAL MODEL AND BOUNDARY CONDITIONS

Flint provides a number of menus for entering and modifying the physical aspects of a CFD problem and the boundary conditions which apply to them. These are:

  • SETUP -> DEFINE PROBLEM
  • SETUP -> PHYSICAL CONSTANTS
  • SETUP -> BOUNDARY-OPTIONS
  • SETUP -> BOUNDARY-CONDITIONS
While these options give users some flexibility in modelling the physical aspects of a CFD problem, as Flint is primarily designed for teaching, an expert CFD engineer will soon find it rather limited in the scope of problems Flint can tackle.
FILE -> SAVE-GRID command will cater for such eventuality allowing the user to save his Flint generated computational-grids into a grid-file
These Grid-Files can later be read into Fluent and used for tackling more complex CFD problems by means of the SETUP - READ-GRID command within Fluent.
